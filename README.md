# Data Structures & Algorithms Applications

A collection of real-world applications developed in **C++** to demonstrate the practical implementation of **Data Structures and Algorithms (DSA)**. This project showcases how fundamental data structures can be used to build efficient and scalable software systems.

---

## 📌 Overview

This repository contains three independent applications built using C++ and the Standard Template Library (STL). Each application focuses on solving a real-world problem while utilizing appropriate data structures to optimize performance.

### Included Applications

- ✅ Smart Task Management System
- ✅ Exam Ranking System
- ✅ Inventory & Billing System

---

## 🚀 Features

### Smart Task Management System
- Priority-based task scheduling using Max Heap
- Fast task lookup using Hash Maps
- Undo functionality using Stacks
- Efficient task organization

### Exam Ranking System
- Student ranking using Min Heap
- Quick record retrieval with Hash Maps
- Automatic rank generation
- Efficient performance analysis

### Inventory & Billing System
- FIFO order processing using Queue
- Item cancellation using Stack
- Low-stock alerts using Min Heap
- Inventory tracking and billing management
